## 소개
개인의 기록을 위한 글 자동완성 블로그 플랫폼

## 기능
- 글 제목을 통한 openai 기반의 글 작성 자동완성
- 게시글의 카테고리 분류
- 글 작성 및 이미지 여러 장 첨부
- 임시 저장 및 수정, 삭제 기능
- 조회수가 가장 많은 게시글 메인보드에 띄우기
